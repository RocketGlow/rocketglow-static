About Us

Our Beliefs

We're passionate about web and the technologies that make it work. We believe in the power of the internet. We believe in efficient rapid development. We believe in dreams. We believe in web standards. We believe in quality. Most of all, we believe in you.

Some of the technologies we work with

HTML5

powering the web like never before

A web framework that was built to bring out innovation and the full potential of the web from a web browser.

CSS3

enhancing the web - style and effect

Cascading Style Sheets add look and feel to web pages. CSS3 has added even better styling and cool stuff like transitions.

jQuery

enhancing the web - style and effect

Multi-browser JavaScript library designed to simplify and speed up client-side scripting, also known famously for its cool transitions.

JavaScript

a new old love

A programming language typically thought of as a browser language that is really starting to gain ground on the server side.

Sass

CSS with superpowers

Sass has more features and abilities than any other CSS extension language out there and is completely compatible with all versions of CSS.

Web Components

custom elements that extend HTML itself

Some of our themes are built strictly for evergreen browsers. We utilize Polymer to make it easier and faster to create these components.

Hammer for Mac

supercharging web development

Hammer saves you time when writing front-end code for static sites, making it easier for you to edit templates with confidence.

Our Hammer Template Themes

Mixture

power of a CMS for static sites

Mixture is from the folk at mixture.io. It's a rapid front-end development toolset for Mac & Windows. Oh, and Mixture is now free!

Our Mixture Boilerplate Themes

Ghost

just a blogging platform

Ghost is a platform dedicated to publishing. It's built on top of Node.js, it's open source and it's free.

Our Ghost Themes
